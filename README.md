# online_code_plagiarism_checker
Check code plagiarism and get instant similarity results.

## Description

A web-based application to detect similarity between code files and generate a plagiarism report with percentage results.

## Features

* Upload code files
* Check similarity
* Display percentage result
* Generate PDF report

## Technologies Used

* Python
* Django
* HTML, CSS
* ReportLab

## How to Run

```bash
git clone https://github.com/your-username/online_code_plagiarism_checker.git
cd online_code_plagiarism_checker
pip install -r requirements.txt
python manage.py runserver
```
